Welcome to the 2000s!

Welcome to an era of the Internet defined by sites with web-safe fonts and colors, ridiculous layouts, and images that act as buttons. It’s the era of GeoCities. It’s the wild west.

So, we can finally start styling websites using CSS. This means we can style text. We can also make use of the box model for all elements. And we can even break the flow of a site using absolute positioning.

Follow this guide to learn how to create a “pleasing” web layout evocative of this era.
