Welcome to the 2010s!

This is an era where the Internet becomes everything. It takes on new functions and allows layouts that appease our every want and need. Perhaps by now, we’ve had too much Internet.

Here’s what sites have now:

  • Advanced layouts using flexbox and CSS grid.
  • Media queries that let us create responsive mobile layouts.
  • Custom fonts.
  • Variable fonts!.
  • All the wonders that JavaScript brings.

For this version of the site, you may NOT use absolute positioning!

If flexbox is confusing, try playing Flexbox Froggy. If CSS Grid is confusing, try playing Grid Garden.
